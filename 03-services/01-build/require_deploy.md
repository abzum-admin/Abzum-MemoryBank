# UC-31 — Abzum ReQuire Tenant Deployment

> **Status: stub.** v1 service. Spec to be filled in alongside the ReQuire v1 build (A75). Full product spec in [`04-products/require/_index.md`](../../04-products/require/_index.md).

## What we deliver

- Provisioned ReQuire tenant for the client
- Recall.ai bot configured for client's Meet/Zoom/Teams workspace
- Voice brain configured (BV: Gemini 3.1 Flash Live; BP: Grok voice-think-fast-1.0)
- Mockup pipeline configured (Claude Artifacts BV / v0 BP / Excalidraw free tier)
- Output schema customised for client's project intake template
- Onboarding training (60-min live)
- 30 days hypercare

## Persona Team

Triage → CEA → Planner → DevOps (per [`05-process/use_case_team_mapping.md`](../../05-process/use_case_team_mapping.md))

## Pricing

Setup fee + monthly hosting + per-discovery-call fee. **Indicative range $2K-$8K setup + $100-$500/month + $20-$50/call.**

## Use Cases for the Client

- Pre-sales discovery for their own customers
- Internal product spec sessions
- User-research interview structuring
