# HR & Talent

> **Status: stub.** Most of what classical HR does in an AI-first company maps to "AI agent lifecycle management."

## Function
**For AI agents** — onboarding new persona disciplines, decommissioning deprecated agents, performance tracking (BI taxonomy A47), capability development roadmap, agent-version migration management. **For humans** — pipeline for the eventual human hires (DM, Principal Engineer, Security Lead, Solution Architect), employment contracts, NZ employment law compliance.

## Capabilities (proposed)
- AI agent activation runbook
- Performance scorecard per persona (using ClickHouse signals + Learning Agent output)
- Human hire pipeline + interview process design
- Employee handbook + culture docs
- Onboarding for new humans
