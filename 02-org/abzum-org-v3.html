Separation of Duties (SoD) Boundaries

Boundary
Agents Separated
Why
Build vs. Run
DevOps ≠ Cloud Platforms Admin
One agent can't control what is deployed AND where it runs
DevSec vs. SOC
Security Agent (CDO) ≠ CSCO Threat Intel / IR
Shift-left DevSec vs. operational security are separate threat models
Access Provisioning
IAM Agent (COO) is sole choke point
No other agent can grant access; CSCO reviews policy only
Spend Approval
Finance & Billing ≠ Procurement
Prevents self-approval of vendor purchases
Compliance Lanes
Legal & Compliance (CFO) ≠ Compliance & Risk (CSCO)
Contractual compliance vs. infosec compliance are distinct domains
Policy vs. Execution
CSCO reviews RBAC; COO/IAM provisions
Policy authority separated from provisioning execution
Build vs. Platform Config
DevOps writes pipelines; Dev Platform Admin owns tooling config
Pipeline author ≠ secrets/repo config owner
