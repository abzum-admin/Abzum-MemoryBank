## Detailed Watchlist

### Paperclip AI ⭐ (Orchestration — PRIMARY)
- **What:** Open-source (MIT) orchestration platform for "zero-human companies"
- **Key:** Companies have org charts, budgets, goals — agents are employees
- **Status:** Primary orchestration backbone for Abzum
- **Superpowers template:** Pre-built 4-agent dev shop

### Superpowers (Workflow Framework)
- **What:** Jesse Vincent's agentic skills framework for human+AI pair programming
- **Key:** TDD, systematic process, YAGNI, evidence over claims
- **Status:** Core workflow framework adapted for Abzum's AI-first model

## Competitive Analysis Framework

When evaluating AI agent products, assess:

| Dimension | Questions |
|---|---|
| **Orchestration** | Does it support multi-agent workflows? |
| **Cost** | Free tier? Per-agent pricing? |
| **Customizability** | Can we modify agent behavior? |
| **Integration** | Does it work with our stack (Azure, OpenClaw)? |
| **MIT/Open Source** | Can we self-host or audit? |
| **Maintenance** | Active development? Community? |

---

## Key Insight

Paperclip AI is the primary orchestration choice (MIT, free, agent-as-employee model). Everything else is tracked for competitive awareness or complementary use cases. Avoid chasing every new product — stay focused on the core stack.
