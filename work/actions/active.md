## Protocol

When work starts on a backlog item:
1. Move from `work/actions/_index.md` to this file
2. Update status to `🔵 In Progress`
3. Record date work started in `Since` column

When work completes:
1. Mark as `✅ Done` in this file
2. Record in `Completed` section
3. Update the master `work/actions/_index.md`
4. Run: `brv curate "Action [ID] completed — [brief description]"`
